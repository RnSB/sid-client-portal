# SID BELKADI | Client Portal Pro (Multi-Language)

Enterprise-grade client portal with neural network background and 3 languages.

## Features

- Neural network animated background (blue on white)
- Multi-language support: English, Français, العربية
- RTL support for Arabic
- Multi-step lead qualification
- AI chat assistant
- Glassmorphism design

## Deploy

1. GitHub → New repo: `sid-client-portal`
2. Upload `index.html`
3. Settings → Pages → Deploy from branch: `main`
4. Live at: `https://username.github.io/sid-client-portal/`

## Language Selector

Users can switch between English, French, and Arabic via the top-right selector. Choice is saved in browser.
